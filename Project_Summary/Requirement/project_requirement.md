# Project Requirement
## INX Future Inc — Employee Performance Analysis | IABAC™

---

## Business Context

INX Future Inc is a leading data analytics and automation solutions provider with over 15 years of global business presence. Consistently rated as a top 20 best employer for the past 5 years.

**Problem:** Employee performance indexes have been declining in recent years.
- Client satisfaction dropped by **8 percentage points**
- Increased service delivery escalations
- CEO concerned about penalising employees without clear data evidence

---

## Project Objectives

The CEO, Mr. Brain, requires the following outputs from this data science project:

| # | Objective |
|---|---|
| 1 | Department-wise performance analysis |
| 2 | Top 3 important factors affecting employee performance |
| 3 | A trained ML model to predict employee performance for hiring |
| 4 | Recommendations to improve employee performance |

---

## Dataset

- **File:** INX_Future_Inc_Employee_Performance_CDS_Project2_Data_V1_8.csv
- **Rows:** 1,200 employees
- **Columns:** 28 features
- **Target:** `PerformanceRating` (2 = Low, 3 = Good, 4 = Excellent)

---

## Deliverables

1. Jupyter notebooks for each stage of the pipeline
2. Trained model saved as `.pkl` file
3. Visualizations of key findings
4. Project summary with business recommendations
